## Description
- 프로젝트 관리를 위한 툴로, 업무를 효율적으로 관리하는데 도움을 줍니다.
- 트렐로(Trello)와 유ㅏ하게 카드 생성, 이동, 삭제를 통해 업무를 관리합니다.

- Vue.js
- npm
